# Node.js Authentication & Authorization (JWT + RBAC)

This project implements a full authentication and authorization workflow using:
- Node.js + Express
- MongoDB + Mongoose
- JWT (role-based access)
- Bcrypt password hashing

## Features
✅ User registration  
✅ Login with JWT  
✅ Protected routes  
✅ Admin-only routes  
✅ Role-based access control
